📁 Manual Cookie Import

When automatic detection doesn't work, you can manually import cookies using browser extensions.

🔧 Recommended Extension: "Get cookies.txt LOCALLY"

📋 Installation Steps:

  1. Chrome: Go to Chrome Web Store → Search "Get cookies.txt LOCALLY"
  2. Firefox: Go to Firefox Add-ons → Search "cookies.txt"
  3. Edge: Go to Edge Add-ons → Search "Get cookies.txt"

📤 Export Process:

  1. Go to youtube.com and log in
  2. Click the extension icon in your toolbar
  3. Select "Export for current site"
  4. Choose "Netscape format"
  5. Save as youtube.com_cookies.txt

📥 Import to Application:

  1. Click "Import Cookie File" button in the main application
  2. Select your saved youtube.com_cookies.txt file
  3. Application will validate and use the cookies automatically

💡 Pro Tips:

  • Export cookies in incognito/private mode for better reliability
  • Re-export cookies every 2-3 days for best results
  • Keep the cookie file in a safe location for reuse
